Gesture Recognition
Using gestures to control applications on your laptop

Gesture Recognition PoC Development

Project Description

The way we interact with a system is changing. We wanted to use gestures to interact with a laptop and perform functions. Additionally, we parallelized the gesture recognition algorithm thereby handling efficiency at the operating system level.

Project Details

  • When: Senior Year | Nov 2014 to December 2014
  • Where: R V College of Engineering | Bangalore
  • Role: Developer
  • Team: Team of 4
  • Time frame: 8 weeks

Problem

People have traditionally used keyboard, mouse or touch for interacting with a laptop. With the advent of new technology, input methods like voice are becoming a trend with Siri on the Macs. This project seeks to explore gestures to control applications on a laptop.

Target Audience

People who like to use different input methods for interacting with a system

Target Solution

This hand gesture recognition algorithm is based on a chapter from the book by K. Kraiss and uses the AForge.Net framework for capturing frames from a webcam in real time. It captures frames using a webcam and processed for detection of the hand. The features are extracted and using the features the gestures are identified which are in turn used to launch different applications. This program was designed in C# with .Net Framework 4 and uses threading support to implement a parallel version of the algorithm.
